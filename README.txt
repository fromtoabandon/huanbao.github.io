本压缩包可直接上传到 GitHub Pages 仓库根目录使用。

包含文件：
1. index.html（首页）
2. organization.html（组织架构）
3. history.html（部门历史）
4. division.html（部门分工）
5. 五个小组独立页面
6. styles.css（公共样式）
7. .nojekyll（GitHub Pages 兼容文件）

说明：
- 所有页面已经替换为可直接展示的正式结构，不再依赖图片占位才能成立。
- 页面之间使用相对路径链接，适合直接上传到仓库根目录。
- 如果后续想补充活动照片，可以在对应页面的卡片区域继续替换或增补内容。
